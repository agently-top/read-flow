# Daily Review Skill - 每日精选

**Use when:** 从候选内容池中生成每日精选日报

**Trigger:** Cron 每天 20:00 执行

**Output:** 结构化日报（飞书推送 + Web 展示）

## 栏目结构

1. **今日大事** - 行业重要新闻（3-5 条）
2. **变更与实践** - 技术更新、最佳实践（2-3 条）
3. **安全与风险** - 安全漏洞、风险提示（1-2 条）
4. **开源与工具** - 新开源项目、工具推荐（2-3 条）
5. **洞察与数据点** - 深度分析、数据报告（1-2 条）
6. **主题深挖** - 反复出现的主题聚合（可选）

## 工作流程

```
后端 API (pending 文章) → LLM 分类整理 → 生成日报 → 飞书推送 + 更新状态
```

## 输出格式

```markdown
## 📰 Read Flow 每日精选 - {日期}

### 今日大事
- [文章 1 标题](链接) - 来源
- [文章 2 标题](链接) - 来源

### 变更与实践
- ...

### 开源与工具
- ...

---
🤖 由 Read Flow 自动生成 | [查看详情](Web 链接)
```
