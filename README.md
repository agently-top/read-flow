# Read Flow - 漏斗式阅读工作流

> 信息过载时代，构建属于自己的认知处理系统

## 项目目标

构建一套完整的信息处理漏斗系统：
- **RSS 广泛捕捉** → **FreshRSS 统一聚合** → **Digest 预处理** → **Daily Review 精选** → **人工精读** → **思源知识库沉淀** → **反馈优化**

## 核心特性

- 📰 日报自动推送（飞书/微信）
- 🌐 独立 Web 审阅界面
- 📚 思源笔记知识库集成
- 📊 周刊自动生成
- 🔄 轻量兴趣画像反馈

## 技术栈

| 层级 | 技术选型 |
|------|---------|
| 信息聚合 | FreshRSS |
| 编排层 | OpenClaw |
| 预处理 | Node.js + Qwen 3.5 Plus |
| 知识库 | 思源笔记 + S3 存储 |
| Web 界面 | 待定（brainstorming 中） |
